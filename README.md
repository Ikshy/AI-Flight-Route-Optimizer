# ✈️ AI Flight Route Optimizer

An AI-powered flight route optimization system using:

Genetic Algorithm (GA) for static multi-stop route optimization  
Reinforcement Learning (RL) with a custom Gym environment for dynamic route planning  
 Real-world airport and route data (OpenFlights dataset)  
Visualization using Matplotlib and Plotly  

---

##  Project Overview

This project simulates and optimizes airline routes using real-world airport data.

It demonstrates:

- Graph-based route modeling
- Evolutionary optimization (GA)
- Custom Reinforcement Learning environment design
- Reward shaping for dynamic decision-making
- Route visualization on world maps

The goal is to explore AI-based optimization techniques used in logistics, aviation, robotics, and autonomous systems.
